# CS-PopulationCalculator
Проект, реализазующий моделирование популяции людей при заданных параметрах (количество людей разных возрастов, вероятность рождения ребенка у двух человек, вероятность смерти для человека каждого возраста. Условно люди разделены на группы по тысяче, для каждой из которых ежегодно проводится проверка на рождение детей и смерть. Отрисовывается график общего количества населения, а так же диаграммы для половозростного состава.
В поле population необходимо указывать количество людей в тысячах, в поля Year start и year end - год начала и окончания симуляции соответственно.
